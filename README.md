# SR Portraits & Events

SR Portraits & Events is a full-stack event services marketplace connecting customers with verified vendors.  
The platform supports vendor onboarding, marketplace listings, bookings, payments, settlements, and admin moderation.  

Vendors can manage profiles, products, calendars, and availability, while customers can browse, book, and pay securely.  
Admins oversee vendor approvals, marketplace content, coupons, banners, and revenue settlements.  

Built with a modern, scalable architecture focused on performance, security, and great user experience.
